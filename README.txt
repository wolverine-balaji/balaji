Life Saver Blood Bank – PHP Project

📁 Project Overview:
This is a web-based Blood Bank Management System built using PHP and MySQL.
It allows user registration, donor information storage, upcoming blood drive tracking, and admin panel access.

🚀 Features:
- 3 members access (User, Admin, Hospital)
- Donor Registration Form with validation
- Admin dashboard to view and manage donors, recipients, and blood stock
- Upcoming Hospital Appointments
- Upcoming Blood Donation camp scheduling
- Blood request and status tracking
- Secure session-based login system
- Bubble animation and UI enhancements for submission appreciation

🛠️ Technologies Used:
- PHP (Backend)
- MySQL (Database)
- HTML5, CSS3 (Frontend)
- XAMPP (Localhost server)
- JavaScript (Form validations)

📦 Folder Structure (Sample):
- `receipient.php` → Donor registration page
- `dashboard.php` → user dashboard
- `need.php` → Blood request form
- `status.php` → Check request status
- `admin.php` → Admin panel & blood stock management
- `db_config.php` → Database connection file
- `2.php` → Homepage with feedback
- `hosp.php` → Hospital panel
